\# Smart City IoT Project



\## Description

This project uses ESP32 to collect data from IR, DHT, and Ultrasonic sensors

and uploads real-time data to Firebase Realtime Database.



\## Components Used

\- ESP32

\- IR Sensor

\- DHT11 Sensor

\- Ultrasonic Sensor (HC-SR04)

\- Firebase Realtime Database



\## Features

\- Real-time sensor monitoring

\- Cloud data storage using Firebase

\- Web dashboard using HTML
